Jak přidat píseň (GitHub Pages bez serveru)

  1. Připrav soubor Guitar Pro (.gp, .gp5, .gpx…).
  2. Pojmenuj: 001-nazev-pisne-autor.gpx (číslo je volitelné, ale pomáhá řazení).
  3. Nahraj do složky /songs přes GitHub web: Otevřít nahrávání ↗
  4. Po commitu akce automaticky vytvoří/aktualizuje data/songs.json.

Tip: Chceš vkládání přes formulář? Lze to přes GitHub Issues + Action, která ze žádosti vytvoří PR.
